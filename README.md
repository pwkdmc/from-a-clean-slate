# From a Blank Slate
A responsive one-page website project about overcoming the fear of the blank page, built with pure HTML and CSS.
## Description
This project is a single-page website titled "From a Blank Slate" that explores the psychological challenges of starting new tasks, particularly the fear of a blank page. It provides insights into why this fear occurs and offers practical strategies to overcome it.
The site features a dark theme with yellow accents, a responsive mobile-first layout, and content structured into thematic sections.
## Features
- Fully Responsive Design: Optimized for mobile devices (360px width baseline).
- Semantic HTML5: Proper use of semantic tags for better accessibility and SEO.
- Custom Typography: Uses Inter and EB Garamond fonts for clean, readable text.
- Themed Sections: Alternating dark and bright themed content sections.
- Card-based Layout: Presents advice in visually distinct card components.
- Clean Navigation: Simple header with active state indication.
## Sections
- Header: Logo and navigation menu
- Cover: Hero section with title "From a Blank Slate"
- Introduction: Explains the problem of procrastination
- Reasons: Lists psychological causes of the "blank page fear"
- Solutions: Provides actionable advice with author examples
- Footer: Copyright and social links
